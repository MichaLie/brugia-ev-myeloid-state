# Repository Scope

This repository is designed as a standalone reproducibility package rather than a submission workspace. The included materials are limited to files that support regeneration, inspection, or verification of the public analytical claims.

Included:

- frozen signature definition and supporting derived evidence tables
- public-input GEO files needed for local reruns
- orthogonal perturbation outputs
- final figure assets and figure legends
- repository-local analysis scripts

Excluded:

- manuscript drafts and tracked revision versions
- internal review documents and editorial notes
- literature PDF libraries
- journal-specific packaging files

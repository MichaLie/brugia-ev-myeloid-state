# Repository Scope

This repository is designed as a standalone reproducibility package rather than a writing workspace. The included materials are limited to files that support regeneration, inspection, or verification of the public analytical claims in the associated analysis.

Included:

- fixed signature definition and supporting derived evidence tables
- cached public-input GEO files needed for selected local reruns
- orthogonal perturbation outputs
- final figure assets and figure legends
- repository-local analysis scripts
- public data, derived table, and module-definition documentation

Excluded:

- drafts and tracked revision versions
- working notes and editorial correspondence
- literature PDF libraries
- publication-package files
- non-public data

The repository is intentionally evidence-bounded. The GSE187403 and GSE143170 perturbation checks can be rerun from cached public inputs. Other layers, including the derivation signature, transcription-factor enrichment, miRNA-target and motif testing, TLR projection, external projection, tolerance comparison, and comparator-state analyses, are preserved as fixed evidence tables for claim inspection and figure regeneration.

This distinction should be kept explicit so that users do not mistake the repository for a complete raw-data recreation pipeline for every public dataset.
